C2: Konsola Pythona i pierwsza inspekcja projektu QGIS#

Programowanie w GIS — Ćwiczenia#

Kurs: Programowanie w GIS (QGIS)
Ćwiczenie: C2
Powiązany wykład: W2
Czas: ~90 minut

Cele ćwiczenia#

Po ukończeniu tego ćwiczenia będziesz potrafić:

  • swobodnie korzystać z Konsoli Pythona i edytora skryptów w QGIS,

  • posługiwać się obiektem QgsProject.instance() do inspekcji projektu,

  • pobierać warstwy po nazwie i typie,

  • odczytywać podstawowe właściwości warstwy wektorowej,

  • iterować po obiektach warstwy i odczytywać ich atrybuty.

Przygotowanie#

Przed rozpoczeciem cwiczenia:

  1. Uruchom QGIS.

  2. Wczytaj co najmniej dwie warstwy wektorowe (np. kraje swiata z Natural Earth oraz dowolna warstwe liniowa lub punktowa). https://www.naturalearthdata.com/downloads/110m-cultural-vectors/ https://www.naturalearthdata.com/downloads/110m-cultural-vectors/ https://www.geoportal.gov.pl/dane/panstwowy-rejestr-granic

  3. Upewnij sie, ze warstwy maja rozne typy geometrii.

  4. Otworz Konsole Pythona: Wtyczki -> Konsola Pythona (Ctrl+Alt+P).

  5. Kliknij Pokaz edytor, zeby miec dostep do edytora skryptow.

1. Rozgrzewka — Konsola Pythona#

Zanim napiszemy wlasciwy kod, upewnijmy sie ze Konsola Pythona dziala poprawnie i ze rozumiemy roznice miedzy jej dwoma trybami pracy.

Zadanie 1.1#

W Konsoli Pythona QGIS wpisz i uruchom ponizsze polecenia jedno po drugim (kazde osobno, wciskajac Enter po kazdym):

print('Konsola dziala')
2 + 2
type(iface)
type(QgsProject.instance())

Zaobserwuj jak QGIS wyswietla wyniki w obszarze wyjscia.

Zadanie 1.2#

Przejdz do edytora skryptow (jesli go nie widzisz, kliknij ikone Pokaz edytor).

Wpisz w edytorze:

print('To jest edytor skryptow')
print('Tutaj pisze sie dluzszy kod')
x = 10
y = 20
print(f'Suma: {x + y}')

Zapisz plik jako test.py, nastepnie uruchom przyciskiem Uruchom skrypt lub skrotem Ctrl+Shift+E.

Pytanie 1.2 — W jakim folderze domyslnie proponuje zapisac plik edytor skryptow QGIS? Co sie stanie jesli zamkniesz QGIS bez zapisania skryptu?

2. Inspekcja projektu przez QgsProject#

Zadanie 2.1#

W Konsoli Pythona wpisz ponizszy kod i uruchom go:

projekt = QgsProject.instance()
print(projekt.title())
print(projekt.fileName())
print(projekt.homePath())
print(projekt.crs().authid())

Jesli projekt nie ma tytulu, metoda title() zwroci pusty ciag znakow ''. Ustaw tytul przez Projekt -> Wlasciwosci projektu -> Ogolne, a nastepnie uruchom kod ponownie i zaobserwuj zmiane.

Pytanie 2.1 — Jaka wartosc zwrocila metoda fileName() przed zapisem projektu? Co to oznacza i jak sie nazywa taki stan projektu?

Zadanie 2.2#

Uruchom ponizszy kod i przeanalizuj wynik:

projekt = QgsProject.instance()

p1 = QgsProject.instance()
p2 = QgsProject.instance()

print(p1 is p2)
print(id(p1))
print(id(p2))

Wskazowka: przypomnij sobie pojecie singleton z wykladu.

Pytanie 2.2 — Co oznacza wynik True dla operatora is? Dlaczego QgsProject jest zaprojektowany jako singleton i jakie praktyczne konsekwencje ma to dla pisania skryptow?

Zadanie 2.3#

Sprawdz ile warstw jest zaladowanych w projekcie oraz wylistuj je:

projekt = QgsProject.instance()
warstwy = projekt.mapLayers()

print(type(warstwy))
print(len(warstwy))

for layer_id, layer in warstwy.items():
    print(layer_id, '|', layer.name())

Pytanie 2.3 — Jakiego typu obiektem jest wynik mapLayers()? Czym jest layer_id — jak wyglada i do czego sluzy?

3. Pobieranie i filtrowanie warstw#

Zadanie 3.1#

Pobierz warstwe po nazwie, a nastepnie sprawdz co sie stanie jesli podasz nazwe warstwy ktora nie istnieje:

projekt = QgsProject.instance()

# Zmien 'kraje' na nazwe jednej z Twoich warstw
wynik = projekt.mapLayersByName('kraje')
print(type(wynik))
print(len(wynik))
print(wynik)

# Teraz podaj nazwe ktora nie istnieje
brak = projekt.mapLayersByName('warstwa_ktora_nie_istnieje')
print(brak)

Pytanie 3.1 — mapLayersByName() zwraca liste, a nie pojedynczy obiekt. Dlaczego tak zostalo zaprojektowane to API? Napisz fragment kodu ktory bezpiecznie pobiera pierwsza warstwe o podanej nazwie i nie spowoduje bledu gdy jej nie ma.

Zadanie 3.2#

Wylistuj tylko warstwy wektorowe z projektu, pomijajac rastrowe:

from qgis.core import QgsMapLayer

projekt = QgsProject.instance()

for layer in projekt.mapLayers().values():
    print(layer.name(), '|', layer.type())

Przyjrzyj sie wartosciom layer.type() dla roznych warstw. Nastepnie zmodyfikuj petlę tak, zeby drukowala tylko warstwy wektorowe.

from qgis.core import QgsMapLayer

4. Wlasciwosci warstwy wektorowej#

Zadanie 4.1#

Wybierz jedna ze swoich warstw wektorowych i odczytaj jej podstawowe wlasciwosci. Uruchom ponizszy kod (dostosuj nazwe warstwy):

from qgis.core import QgsWkbTypes

warstwy = QgsProject.instance().mapLayersByName('kraje')  # zmien nazwe
layer = warstwy[0]

print(f'Nazwa          : {layer.name()}')
print(f'Zrodlo         : {layer.source()}')
print(f'CRS            : {layer.crs().authid()}')
print(f'Typ geometrii  : {QgsWkbTypes.displayString(layer.wkbType())}')
print(f'Liczba obiektow: {layer.featureCount()}')
print(f'Liczba pol     : {layer.fields().count()}')

zasieg = layer.extent()
print(f'Zasieg xMin    : {zasieg.xMinimum():.4f}')
print(f'Zasieg xMax    : {zasieg.xMaximum():.4f}')
print(f'Zasieg yMin    : {zasieg.yMinimum():.4f}')
print(f'Zasieg yMax    : {zasieg.yMaximum():.4f}')

Pytanie 4.1 — Co oznacza wartosc zwrocona przez layer.source()? Czym rozni sie source() od fileName() w kontekscie warstwy?

Zadanie 4.2#

Wylistuj wszystkie pola atrybutow wybranej warstwy wraz z ich typami.

layer = QgsProject.instance().mapLayersByName('kraje')[0]  # zmien nazwe

pola = layer.fields()
print(f'Liczba pol: {pola.count()}')
print()

for pole in pola:
    print(f'  {pole.name():<20} {pole.typeName():<12} dlugosc: {pole.length()}')

Nastepnie odpowiedz na pytanie ponizej.

Pytanie 4.2 — Znajdz w wyniku pole przechowujace liczbe ludnosci (lub inne pole numeryczne). Jaki ma typ? Czym rozni sie typ Integer od Real w kontekscie atrybutow GIS?

Zadanie 4.3 — Samodzielne#

Bez korzystania z gotowego kodu z wykladu, napisz w edytorze skryptow fragment ktory dla aktywnej warstwy wypisuje:

  • nazwe warstwy,

  • kod EPSG jej ukladu wspolrzednych,

  • typ geometrii jako czytelny napis (np. 'Polygon'),

  • liczbe obiektow,

  • liste nazw wszystkich pol atrybutow w jednej linii oddzielonych przecinkami.

Wskazowka: uzyj iface.activeLayer() zeby pobrac aktywna warstwe. Zaznacz warstwe w Panelu warstw i uruchom skrypt.

5. Iteracja po obiektach i atrybutach#

Zadanie 5.1#

Uruchom ponizszy kod na warstwie wektorowej i przeanalizuj wynik:

layer = QgsProject.instance().mapLayersByName('kraje')[0]  # zmien nazwe

print('Pierwsze 3 obiekty:')
print()

for i, obiekt in enumerate(layer.getFeatures()):
    if i >= 3:
        break
    print(f'id         : {obiekt.id()}')
    print(f'atrybuty   : {obiekt.attributes()}')
    print(f'typ geom   : {obiekt.geometry().type()}')
    print()

Pytanie 5.1 — obiekt.attributes() zwraca liste wartosci. Jak odniesiesz konkretna wartosc do nazwy pola? Napisz dwa sposoby dostepu do wartosci pola o nazwie 'name': jeden przez indeks, drugi przez nazwe pola.

# Twoja odpowiedz i dwa sposoby dostepu:

Zadanie 5.2#

Policz ile obiektow warstwy ma wartosc w pewnym polu numerycznym wiekszą od progu ktory sam/a ustalisz. Na przykład, liczba krajów w populacji powyżej 10 mln ludzi. Uzupelnij ponizszy szablon:

layer = QgsProject.instance().mapLayersByName('kraje')[0]  # zmien nazwe

POLE = 'pop_est'    # zmien na nazwe pola numerycznego w Twojej warstwie
PROG = 10_000_000   # zmien na dowolna wartosc

licznik = 0
for obiekt in layer.getFeatures():
    wartosc = obiekt[POLE]
    if wartosc is not None and wartosc > PROG:
        licznik += 1

print(f'Obiektow z {POLE} > {PROG:,}: {licznik}')
print(f'Wszystkich obiektow      : {layer.featureCount()}')

Pytanie 5.2 — Dlaczego w warunku sprawdzamy wartosc is not None zanim porowniemy ja z progiem? Co sie stanie jesli tego nie zrobimy i w warstwie jest obiekt z pusta wartoscia w danym polu?

Zadanie 5.3#

Uruchom ponizszy kod i zaobserwuj roznice w czasie wykonania:

import time
from qgis.core import QgsFeatureRequest

layer = QgsProject.instance().mapLayersByName('kraje')[0]  # zmien nazwe
POLE = 'continent'
WARTOSC = 'Europe'

# Sposob 1: filtrowanie w Pythonie
t0 = time.time()
wynik1 = [ob for ob in layer.getFeatures() if ob[POLE] == WARTOSC]
t1 = time.time()
print(f'Filtrowanie Python : {len(wynik1)} obiektow w {t1-t0:.4f} s')

# Sposob 2: filtrowanie przez QgsFeatureRequest
t0 = time.time()
req = QgsFeatureRequest().setFilterExpression(f'"{POLE}" = \'{WARTOSC}\'')
wynik2 = list(layer.getFeatures(req))
t1 = time.time()
print(f'QgsFeatureRequest  : {len(wynik2)} obiektow w {t1-t0:.4f} s')

Pytanie 5.3 — Ktory sposob jest szybszy i dlaczego? Przy jakiej wielkosci warstwy roznica staje sie istotna w praktyce? Kiedy mimo to mogloby byc lepiej filtrowac w Pythonie?

6. Zadanie laczace#

Ponizsze zadanie laczy umiejetnosci z calego cwiczenia. Napisz je samodzielnie w edytorze skryptow QGIS, a nastepnie wklej kod ponizej.

Zadanie 6.1 — Skrypt statystyk warstwy#

Napisz skrypt ktory dla aktywnej warstwy wektorowej wyswietla:

  1. Nazwe warstwy, typ geometrii i liczbe obiektow.

  2. Liste pol atrybutow z ich typami.

  3. Dla kazdego pola numerycznego — minimalna i maksymalna wartosc wsrod obiektow (iteruj po obiektach i szukaj min/max recznie, bez uzycia bibliotek statystycznych).

  4. Komunikat bledu jesli aktywna warstwa nie jest wektorowa lub jesli brak aktywnej warstwy.

Wymagania dotyczace kodu:

  • uzyj iface.activeLayer(),

  • sprawdz typ warstwy przed iteracja,

  • pomijaj wartosci None przy obliczaniu min/max,

  • typ pola numerycznego to 'Integer', 'Real' lub 'Integer64'.
